# tf-module-mysql

All the codes needed to create the RDS [A Managed service for SQL on AMS] is going to be placed in this backend module and the associated root module will be 'terraform-databases' 

DB's supported by MYSQL :

```
1) MariaDB
2) SQL-server
3) Oracle
4) Aurora
5) Postgress
6) Aurora Postgress
7) IBM DB 2
8) Microsoft sql

```
